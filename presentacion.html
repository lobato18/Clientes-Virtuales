SECTION: 09_VNC_SECURITY // ACCESSIBILITY

Seguridad y Conexión Remota VNC_TUNNEL

  • Conexión Gráfica VNC: Usamos clientes gráficos estándar (vncviewer, TigerVNC, Remmina) para un control directo de los escritorios virtuales.
  • Túnel SSH para VNC: Ciframos el puerto de comunicación gráfico (5900+ de la VM) para evitar fugas de contraseñas de sesión.
  • Llaves RSA/ED25519: Eliminamos contraseñas básicas para el acceso directo a la consola del host físico.
